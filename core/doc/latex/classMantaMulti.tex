\subsection{\-Detailed \-Description}
\-Superclass that adds functionality for multiple access to a single \hyperlink{classManta}{\-Manta}. 

\-This class can be used by an application that wants to have multiple \-Manta\-Clients that connect to a single \hyperlink{classManta}{\-Manta} using the \hyperlink{classMantaServer}{\-Manta\-Server} \-A\-P\-I. \-If you only need single-\/access you should use the \hyperlink{classManta}{\-Manta} class instead
