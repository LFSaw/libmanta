\subsection{\-Detailed \-Description}
\-Superclass that provides an interface to the \hyperlink{classManta}{\-Manta}. 

\-The \hyperlink{classManta}{\-Manta} class is intended to use as a base class within your application. \-In order to use libmanta, subclass this and define implementations for any callback functions that you would like your application to be notified of. \-The event functions are declared in the \hyperlink{classMantaClient}{\-Manta\-Client} interface.

\-Creating an instance of the \hyperlink{classManta}{\-Manta} class (or a subclass) will not initiate a connection to any plugged-\/in mantas. \-Before you can start communicating with the \hyperlink{classManta}{\-Manta} you must call its \hyperlink{classMantaUSB_a51e68daa5edfbc7ced89b6f9faa86ed5}{\-Connect()} method to connect to a physical \hyperlink{classManta}{\-Manta} over \-U\-S\-B.

\-Once connected, your application should periodically call the static method \hyperlink{classMantaUSB_a4d55c1c1d9e769ce7a0ef532a72221e8}{\-Handle\-Events()}. \-This will take care of servicing the low-\/level \-U\-S\-B communication for all connected \-Mantas.

   // Algorithm from Bit Twiddling Hacks
   uint8_t outByte = inByte; // first get LSB of inByte
   int s = 7; // extra shift needed at end

   for (inByte >>= 1; inByte; inByte >>= 1)
   {   
      outByte <<= 1;
      outByte |= inByte & 1;
      s--;
   }
   outByte <<= s; // shift when inByte's highest bits are zero
   return outByte;
